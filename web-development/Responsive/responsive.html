Grundlagen

Responsive Webdesign bedeutet, dass sich eine Webseite automatisch an verschiedene Bildschirmgrößen anpasst.

Media Queries

Mit Media Queries können verschiedene CSS-Regeln für unterschiedliche Bildschirmgrößen definiert werden.

Flexbox

Flexbox ist eine CSS-Technik zur einfachen und flexiblen Anordnung von Elementen.

CSS Grid

CSS Grid ist ein leistungsstarkes Layout-Tool, das sich hervorragend für komplexe Designs eignet.

Responsive Bilder

Bilder sollten sich dynamisch an verschiedene Bildschirmgrößen anpassen, um eine optimale Darstellung zu gewährleisten.
